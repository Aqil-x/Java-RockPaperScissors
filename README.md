# Java-RockPaperScissors
This is a small simple rock, paper, scissors game to show my skills using java. It is a simple and small game that does not require much code but as someone who wants to grow in skill i believe this shows basic elements that are used in Java.

Features
Play against the computer
User input via Java Scanner
Random computer choices using Random
Clear result messages
Beginner-friendly structure
Easy to run and understand

How It Works
The user picks Rock (1), Paper (2), or Scissors (3)
The computer randomly generates a number between 1 and 3
The program compares choices and determines the winner
Results of the winner are displayed

What I Learned
How to use the Scanner class for input
How to generate random numbers with Random
Using methods to organize code
Using switch for mapping numbers to text
Basic conditional logic in Java
